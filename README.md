Install dependencies:

```
npm install
```

Set environment variables:

```
cp .env.example .env
```

## Running Locally

```
npm run dev
```

## Running in Production

```
npm start
```

## Lint

```
# lint code with ESLint
npm lint
```
```
# try to fix ESLint errors
npm lint:fix
```
```
# lint and watch for changes
npm lint:watch
```

## Documentation

```bash
# generate and open api documentation
npm docs
```
