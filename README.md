Install dependencies:

```
npm install
```

Set environment variables:

```
cp .env.example .env
```

## Running Locally

```
yarn dev
```

## Running in Production

```
yarn start
```

## Lint

```
# lint code with ESLint
yarn lint
```
```
# try to fix ESLint errors
yarn lint:fix
```
```
# lint and watch for changes
yarn lint:watch
```

## Documentation

```bash
# generate and open api documentation
yarn docs
```
